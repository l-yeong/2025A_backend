[ 자바 구동 원리 ]
    1. [코딩] 개발자는 .java 파일 내 자바 문법 작성한다.
    - Example1.java (파일)
    2. [실행].java 파일 ---> 컴파일(번역/해석) ---> .class 파일 생성
    3. [처리] (컴퓨터본체).class 처리 한다.

[ 자바 활용처 ]
    1. 소프트웨어 : javafx툴 활용, 등등
    2. 웹 : JSP, Spring 등등
    3. 앱 : 안드로이드 등등
    (자바언어/.java)

[ 자바 코드 실행 ]
    자바는 클래스 단위로 컴파일 되고, 컴파일된 코드는 main함수 부터 읽어서(스레드) 실행 된다.
